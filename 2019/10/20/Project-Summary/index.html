Project-Summary

What is my project?

Using Python programming to help to decide how to group different breast cancer patients in order to use more specific method to cure.

What are my research target and research method?

Using Python programming to help to decide how to group different breast cancer patients in order to use more specific method to cure. The major grouping method is PCA.

What was the biggest challenge that I encountered? How did I overcome it?

Deciding which grouping method fits my project the most and which factors I should consider. I try to overcome them by understanding different grouping methods and knowing their characteristics.

What are my research result(s) and conclusion(s)?

Based on the data of PAM50 mRNA and other possible categories, by using kmeans and PCA to categorize different groups is a reasonable categorizing way.
